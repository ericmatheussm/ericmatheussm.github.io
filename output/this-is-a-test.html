Do you know how to obtain the value of all the \(nth\) roots a number? For example, how do we obtain all the 5 values of \(r\) that satisfy \(r = \sqrt[5]{4}\) ?

According to Wikipedia, the "nth root of a number \(x\), where \(n\) is a positive integer, is any of the \(n\) real or complex numbers \(r\) whose nth power is \(x\)":

$$ r^n = x $$

Today, when studying about complex numbers, I found a way of obtaining each one of these values. We start by verifying that \(x=x\cdot e^{j2\pi k}\), for any integer \(k\).

$$ x\cdot e^{j2\pi k} = x\cdot\cos(2\pi k) + j\cdot x \cdot \sin(2\pi k) = x +j\cdot x \cdot 0 = x $$

Therefore,

$$ \sqrt[n]{x} = x^{\frac{1}{n}} = x^{\frac{1}{n}}e^{j2\pi k/n} $$

for \(k = 0, 1, 2, 3, \ldots, n-1\)
